# Archive

Superseded roadmaps, research, reviews, and plans. **One rule: this directory
stays flat — no nesting, ever.** Filenames are dated; the directory doesn't
need to re-encode age.

Lifecycle: when a roadmap in `../roadmaps/` is superseded, move it here and
leave a one-line tombstone in the Shortlist pointing at the replacement.
That's the whole ceremony. Git history preserves everything else.

Live documents stay out of here: `../roadmaps/` (current plans + Shortlist),
`../MM-DD-YY-Summary/` (dated dev summaries), `../user-guide/`.
